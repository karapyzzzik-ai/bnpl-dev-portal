# Авторизация

### 1. Метод авторизации для получения Bearer токена

Метод: **POST**

##### Пример запроса

```
curl --request POST \
  --url https://dev.bnpl.kz/api/users/api/v1/sign-in \
  --header 'Content-Type: application/json' \
  --header 'accept: application/json' \
  --data '{
"password": "password",
"username": "username"
}'
```

_Если интеграция происходит на stage среде, запрос необходимо отправлять на адрес https://stage.bnpl.kz/api/users/api/v1/sign-in_

##### Структура запроса

| Параметр | Тип | Обязательно | Описание | Правила валидации |
| --- | --- | --- | --- | --- |
| username | string | да | Логин | количество символов от 6 до 100. |
| password | string | да | Пароль | количество символов от 6 до 100. |
