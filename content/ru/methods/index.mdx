# Методы

Обзор методов API для интеграции с bnpl.kz.

- [Авторизация](/ru/methods/auth) — получение и обновление Bearer-токена.
- [Скоринг](/ru/methods/scoring) — создание заявки и получение результатов скоринга.
- [Изменение статуса заказа](/ru/methods/change-status/delivery) — доставка, частичная доставка, отмена, возврат.
- [Статус заказа](/ru/methods/preapp) — получение текущего статуса заявки/заказа.
