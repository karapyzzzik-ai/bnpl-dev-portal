# Получение лимита на рассрочку по номеру телефона

Метод предназначен для партнёров и позволяет получить по номеру телефона клиента сумму доступного лимита на рассрочку.

| | |
| --- | --- |
| Endpoint | `/v1/limit/{phone_number}/installment-limit` |
| Тип HTTP метода | GET |
| Формат данных | JSON |

### Авторизация

Bearer

#### Запрос

path параметр

| Параметр | Тип | Обязательно | Правила валидации | Пример | Описание |
| --- | --- | --- | --- | --- | --- |
| phoneNumber | string | Да | Соответствие формату | 7XXXXXXXXXX | Номер телефона клиента |

#### Ответ

Пример ответа HTTP 200

```json
{
"installmentLimit": 37000
}
```

Структура ответа

| Название | Тип данных | Описание |
| --- | --- | --- |
| installmentLimit | int | Сумма доступного лимита |

HTTP 400 (ошибка - в запросе отсутствует поле phoneNumber)

```json
{
"code": 0,
"message": "string",
"details": [
    {
      "@type": "string",
      "additionalProp1": "string",
      "additionalProp2": "string",
      "additionalProp3": "string"
    }
]
}
```

#### Статусы

Статусы ответа метода

| Статус | Описание |
| --- | --- |
| 200 | При успешном выполнении метода |
| 400 | При клиентской ошибке. (Пример: неверный формат phone_number) |
| 401 | Ошибка авторизации |
| 404 | При отсутствии клиента с таким номером телефона |
| 500 | Общая ошибка сервера |
