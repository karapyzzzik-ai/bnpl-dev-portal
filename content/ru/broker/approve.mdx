# Отправка выбора клиента

| Метод | POST |
| --- | --- |
| URL | `bnpl/broker/v1/approve` |
| Описание | Получение выбора от клиента |
| Формат ответа | JSON |
| Статус ответа | 200 — В случае успешного сценария |
|  | 400 — В случае ошибки |
|  | 500 — Неизвестная ошибка |

## Запрос

```json
{
"id": "uuid"
"code": "BNPL4"
}
```

### Структура

| Параметр | Тип | Описание | |
| --- | --- | --- | --- |
| id | | uuid | Идентификатор BNPL |
| code | | string | Код продукта BNPL |
| Authorization | header | Bearer token | Авторизация |

### Успешный ответ

```json
{
       "redirectLink": "string"
}
```

На полученный redirectLink необходимо перенаправить клиента

### Отрицательный ответ

```json
{ "error": { "code": int, "message": "string" } }
```
