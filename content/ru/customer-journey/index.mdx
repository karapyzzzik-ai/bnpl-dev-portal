По нажатию кнопки "Перейти к оплате" партнер отправляет на Backend BNPL пакет данных для создания предзаявки

Партнер перенаправляет клиента по полученной ссылке (redirectLink)

##### Пример redirectLink на внешнем тестовом стенде

```
https://dev.bnpl.kz/scoring/?preapp=ec162f55-e0a9-5f89-fa39-2720c8fc80a2&phone=&link=https%3A%2F%2Fwww.flip.kz
```

### Виджет (Модальное окно)

**Виджет** (в рамках понятии продукта BNPL) – js библиотека, которая позволяет оформить продукт на странице чекаута мерчанта, без редиректа клиента в наш UI. С демо можно ознакомиться тут.

_\*Мы рекомендуем использовать данный метод для бесшовной интеграции._
