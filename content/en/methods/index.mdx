# Methods

Overview of the API methods for integrating with bnpl.kz.

- [Authorization](/en/methods/auth) — obtaining and refreshing a Bearer token.
- [Scoring](/en/methods/scoring) — creating an application and getting scoring results.
- [Changing order status](/en/methods/change-status/delivery) — delivery, partial delivery, cancellation, return.
- [Order status](/en/methods/preapp) — getting the current status of an application/order.
