# Sending the client's choice

| Method | POST |
| --- | --- |
| URL | `bnpl/broker/v1/approve` |
| Description | Getting the choice from the client |
| Response format | JSON |
| Response status | 200 — On successful scenario |
|  | 400 — On error |
|  | 500 — Unknown error |

## Request

```json
{
"id": "uuid"
"code": "BNPL4"
}
```

### Structure

| Parameter | Type | Description | |
| --- | --- | --- | --- |
| id | | uuid | BNPL identifier |
| code | | string | BNPL product code |
| Authorization | header | Bearer token | Authorization |

### Successful response

```json
{
       "redirectLink": "string"
}
```

The client must be redirected to the received redirectLink

### Negative response

```json
{ "error": { "code": int, "message": "string" } }
```
